# 因果核 三项目对比

**数据路径**：`src/task/novel/outputs/`  
**对比项目**：因果核_同大纲_关云端 · 因果核_同大纲_开云端 · 因果核_新大纲_关云端  
**对比范围**：第 21–25 章 大纲 + 正文

因果核_同大纲_关云端 由「复制 因果核_同大纲_开云端 → 删除 21–25 章文件 → 关云端续写 21–25」得到，与同大纲_开云端 为**同一大纲**，仅续写时是否注入云端记忆不同。

---

## 总览


| 项目          | 21–25 大纲线                | 续写  | 第21章要点            | 第25章要点                |
| ----------- | ------------------------ | --- | ----------------- | --------------------- |
| **同大纲_开云端** | 北境之外→灵路→西荒风沙→沈无妄→禁地残响    | 开云端 | 共鸣玉、东极、第七层遗言、小满光纹 | 禁地、备份林渊、第七层/共鸣玉、第7次迭代 |
| **同大纲_关云端** | **同上**（北境之外→…→禁地残响）      | 关云端 | 重铸、离开北境、共感链接、九大灵脉 | 禁地、玉简石板、第七次迭代、沈无妄     |
| **新大纲_关云端** | 离北之计→西荒残照→灭门之溯→中州风起→顾氏长歌 | 关云端 | 重铸/归零、去中州、灵脉光幕    | 顾氏长歌、7号变量、天演模型、必死局    |


同大纲_开 vs 同大纲_关 = **同大纲、仅续写开/关云端**，可单独看云端记忆对正文与伏笔的影响；新大纲_关 = 另一套大纲（中州线），第 25 章为顾氏长歌。

## 实验设计


| 项目名             | 大纲 21–25                         | 续写时   | 作用                           |
| --------------- | -------------------------------- | ----- | ---------------------------- |
| **因果核_同大纲_开云端** | 北境之外 → 灵路算法 → 西荒风沙 → 沈无妄 → 禁地残响  | 开云端记忆 | 长程召回参与续写（基线）                 |
| **因果核_同大纲_关云端** | **与上相同**（北境之外→…→禁地残响）            | 关云端记忆 | 仅本地 mesh + 摘要，无 EverMemOS 注入 |
| **因果核_新大纲_关云端** | 离北之计 → 西荒残照 → 灭门之溯 → 中州风起 → 顾氏长歌 | 关云端记忆 | 另一套大纲（中州线），关云端续写             |


- **核心对比**：同大纲_开云端 vs 同大纲_关云端 → **同一大纲**，唯一变量为续写时是否注入云端记忆，可观察长程召回对伏笔回指与细节一致性的影响。
- **新大纲_关云端**：与上两套大纲不同，第 25 章已到中州/顾长歌，用于对比「不同大纲 + 关云端」的走向。

| 项目          | 开篇 / 核心要素                                                                      |
| ----------- | ------------------------------------------------------------------------------ |
| **同大纲_开云端** | 偏院月洞门、苏晚已离开；小满送**共鸣玉**；回指「三天前藏书阁」「第七层遗言」；苏晚传讯「代码」苏醒、去**东极**；小满能见「光纹」、茶馆探子事件。   |

## 结论


| 对比维度               | 要点                                                                                                       |
| ------------------ | -------------------------------------------------------------------------------------------------------- |
| **同大纲_开 vs 同大纲_关** | 同一套大纲（北境之外→…→禁地残响），唯一变量为续写时**是否注入云端记忆**。开云端版前文回指更密（共鸣玉、东极、藏书阁、小满光纹、备份林渊等）；关云端版主线一致，细节与道具回指较少，更偏当前章对话与设定。 |
| **新大纲_关**          | 另一套大纲（离北之计→…→顾氏长歌），第 25 章已到中州/顾长歌。与同大纲两版对照，体现「大纲扩展」不同（北境/西荒/禁地 vs 中州线）带来的情节分流。                           |


**小结**：在**同大纲**前提下，续写时开启云端记忆（EverMemOS）能明显增强多章续写中的**伏笔与细节回指**；新大纲_关则说明大纲阶段的选择会直接决定后续走向（禁地线 vs 中州线）。
