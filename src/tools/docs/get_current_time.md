***REMOVED*** get_current_time

**描述**: 获取指定时区的当前时间和日期

***REMOVED******REMOVED*** 参数

- `timezone` (string)（可选）: 时区名称，例如：'Asia/Shanghai'、'America/New_York'、'UTC'。默认为 'Asia/Shanghai'
- `format` (string)（可选）: 时间格式，'full' 表示完整格式，'date' 表示仅日期，'time' 表示仅时间。默认为 'full'

***REMOVED******REMOVED*** 使用示例

```json
{
  "timezone": "示例值",
  "format": "示例值"
}
```
